# Quick Start Guide

## For End Users

### Windows

1. Download `DFGviz-windows-v1.0.0.zip` from the [Releases](https://github.com/SuzaneANO/DFGviz/releases) page
2. Extract the ZIP file
3. Double-click `ScalpelDataflowAnalyzer.exe`
4. Select your git repository and start analyzing!

### Linux

1. Download `DFGviz-linux-v1.0.0.tar.gz` from the [Releases](https://github.com/SuzaneANO/DFGviz/releases) page
2. Extract the archive: `tar -xzf ScalpelDataflowAnalyzer-linux-v1.0.0.tar.gz`
3. Run: `./ScalpelDataflowAnalyzer`
4. Select your git repository and start analyzing!

### macOS

1. Download `DFGviz-macos-v1.0.0.tar.gz` from the [Releases](https://github.com/SuzaneANO/DFGviz/releases) page
2. Extract the archive: `tar -xzf ScalpelDataflowAnalyzer-macos-v1.0.0.tar.gz`
3. Run: `./ScalpelDataflowAnalyzer`
4. Select your git repository and start analyzing!

## Trial Period

The application includes a **30-day trial period**. After installation, you'll see a message showing how many days remain. After 30 days, you'll need to purchase a license to continue using the software.
